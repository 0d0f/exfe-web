Front-End Standards 前端标准规范
--------------------------------

## Avatar:
  * avatar size >= 40: border-radius(4px)
  * avatar size < 40: border-radius(2px)

## Identity:
  * 根据上下文，替换 __provider name__, __provider icon__
  * provider
    - email
    - twitter
    - facebook
    - ...

## UI:

### Dialogs:
  * Button
    - class="disabled"
      - opacity(50%)
